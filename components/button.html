# Button 按钮

常用的操作按钮。优先推荐使用与本主题风格统一的 plain 样式。

# 基础样式

使用type、plain和round属性来定义 Button 的样式。

增加 type='dark' 类型

<el-button type="light" plain>浅色按钮</el-button>
<el-button type="dark" plain>深色按钮</el-button>

# Glow 效果

未修改 Element 组件， 因此通过引用 is-glow 样式实现。

<el-button class="is-glow">Glow 按钮</el-button>

# 禁用状态

按钮不可用状态。

你可以使用disabled属性来定义按钮是否可用，它接受一个Boolean值。

# 文字按钮

没有边框和背景色的按钮。

# 按钮组

以按钮组的方式出现，常用于多项类似操作。

使用<el-button-group>标签来嵌套你的按钮。

# 加载中

# 不同尺寸

Button 组件提供除了默认值以外的三种尺寸，可以在不同场景下选择合适的按钮尺寸。

额外的尺寸：medium、small、mini，通过设置size属性来配置它们。
